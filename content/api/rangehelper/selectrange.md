## selectRange()

{{% api_method name="selectRange(range)" since="1.3.5" %}}

Selects the passed DOM range.


{{% api_section title="Syntax" %}}
```js
instance.selectRange(range);
```
{{% /api_section %}}


{{% api_parameters %}}
{{% api_parameter %}}
**range**  
Type: **[Range](/api/types/#range)** or **[TextRange](/api/types/#textrange)**

The range to select.
{{% /api_parameter %}}
{{% /api_parameters %}}
